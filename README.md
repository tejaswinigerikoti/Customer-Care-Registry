# Customer Registry System

A full-stack MERN (MongoDB, Express.js, React.js, Node.js) application for managing customers, complaints, and support agents.

---

## Features

- User Authentication
- Dashboard
- Customer Management (CRUD)
- Complaint Management (CRUD)
- Agent Management (CRUD)
- Reports Dashboard
- Pie Chart & Bar Chart Analytics
- PDF Report Export
- Responsive Bootstrap UI

---

## Tech Stack

### Frontend
- React.js
- Bootstrap 5
- Axios
- Chart.js
- jsPDF

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication

---

## Installation

### Clone Repository

```bash
git clone <repository-url>
```

### Backend

```bash
cd server
npm install
npm start
```

### Frontend

```bash
cd client
npm install
npm run dev
```
